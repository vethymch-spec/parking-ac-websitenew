# 驻车空调网站设计方案

## 选定方案：工业科技风 × 户外冒险美学

**设计运动**：工业现代主义 + 户外探险美学（Industrial Modernism × Outdoor Adventure Aesthetic）

**核心原则**：
- 专业可信赖：通过深蓝色调和精密排版传达技术专业感
- 户外自由感：大幅场景图片唤起露营旅行的自由氛围
- 清晰的产品层级：产品展示区采用左右交替布局，突出每款产品特性
- 功能性美学：设计服务于转化，按钮层级分明

**色彩哲学**：
- 主色：深海蓝 #0f3460 → 传达专业、可靠
- 强调色：电蓝 #1a73e8 → CTA按钮，充满活力
- 背景：浅冰蓝 #e8f4fd → 产品区域，清爽干净
- 深色区：#1e3a5f → 页脚和Shop with Us区域
- 公告栏：#0e6b8a 深青色

**布局范式**：
- 全屏英雄区（暗色调户外场景）
- 产品展示：左右交替图文布局（50/50分割）
- 特性展示：同样左右交替，图片+文字
- 4列优势卡片区（深蓝背景）
- 3列博客卡片

**标志性元素**：
- 顶部细公告栏（深青色背景）
- 产品标签徽章（COOLING / HEATING 等）
- 实心蓝色CTA按钮 vs 描边按钮的层级对比

**交互哲学**：
- 导航悬停下划线动效
- 按钮悬停颜色加深
- 图片区域轻微缩放效果

**动画**：
- 页面滚动进入动画（fade-in + slide-up）
- 导航栏滚动后背景变化
- 按钮hover过渡 200ms ease

**字体系统**：
- 标题：Montserrat Bold/ExtraBold（力量感）
- 正文：Inter Regular（可读性）
- 小标签：Montserrat Semibold 大写字母
